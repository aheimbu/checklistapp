### \# Checklistapp



###### \## Description

GUI application to manage checklists saved in user profiles.

User data is saved in local database.





###### \## User stories

\### prio high

\[ ] As a user, I want to create an account so that my checklists are saved privately.

\[ ] As a user, I want to log in so that I can access my checklists.

\[ ] As a user, I want to log out so that my account is secure on shared devices.

\[ ] As a user, I want to create a checklist so that I can organize related tasks.

\[ ] As a user, I want to delete a checklist so that I can remove lists I no longer need.

\[ ] As a user, I want to add items to a checklist so that I can track individual tasks.



\### prio mid

\[ ] As a user, I want to assign a quantity to a checklist item so that I can plan how much I need.

\[ ] As a user, I want to mark checklist items as completed so that I can see my progress.

\[ ] As a user, I want to edit checklist items so that I can correct mistakes.

\[ ] As a user, I want to export a checklist as a text file, so that I can copy it and paste it for my needs.



###### \## Database

Using MySQL Workbench + Server.
